PokeHexa is a open source Discord bot coded in JavaScript with Telegraf by [RedAura](https://github.com/MoeZilla).  
```
- `npm install`
- Put your GBA rom named `hexa.gba` in `./`
- Put your GBA save named `hexa.sav` in `./`
- Prepare your `.env` file

BOT_TOKEN=<not token>
  
```
